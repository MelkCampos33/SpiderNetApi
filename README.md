# 🕸️ Lista de Exercícios – Sistemas Distribuídos
## 🕷️ SpiderNet API – Registro de Missões do Homem-Aranha

---

## 📌 Orientações

- A atividade pode ser realizada **individualmente ou em equipes de até 5 alunos**
- Entrega **até a data definida em aula**
- Desenvolver em **C# com .NET 10**
- Seguir padrão **MVC (Model–View–Controller)**
- Código deve estar **organizado e indentado**
- Projeto deve estar no **GitHub pessoal**
- Entregar **apenas a URL do repositório**

---

## 🧠 Contexto

Após os eventos do multiverso, **Peter Parker** continua atuando como o Homem-Aranha em Nova York e precisa de um sistema para organizar suas missões.

Para ajudar, **Ned Leeds** criou a API distribuída **SpiderNetApi**, permitindo:

- Registrar novas missões  
- Consultar histórico de atividades  

Essa API poderá ser usada por:

- Polícia  
- Vingadores  
- Traje Stark
